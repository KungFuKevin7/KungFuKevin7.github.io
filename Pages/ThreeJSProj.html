Biography Game


Assignment Info

For this solo-project i wanted to create a minigame where you could drive a car over a motorway, whilst learning more about me. I wanted it to have a very cartoonish style to it, hence most objects do not feature a texture. For this project i wanted to use Three.js, since i had some experience with it and because i wanted to further improve my javascript skills.
Did you already give it a try? no? Click here to try it out!


My tasks

I had to create all objects i wanted to feature in my world, including: the car, road signs, trees and lanterns. I also had to create the scene where all objects were placed, the camera and the light source were initialized. Next to that, i had to create the controls and convert user input into car movement. Lastly, i had to take photos and draw textures to display in my scene.


What have i learnt?

This project taught me how to program using javascript, along with the three.js library. I also learnt how to create an interactive minigame using three.js. Lastly i also learnt how Computer Graphics work (mathematical side).
