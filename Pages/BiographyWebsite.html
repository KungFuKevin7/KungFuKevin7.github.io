My Personal Website


HTML CSS Javascript Three.JS

Assignment Info

I wanted to create my very own website, because i felt it was a sin not having one as a programmer/coder. With this website i wanted to show off my skill in programming/coding and just to keep on programming during the summer break. I also wanted my website to be more interactive than any other, so i thought to myself: "Why not implement a minigame?". And thus the idea to create my very own Biography Game was born.


My tasks

  • Designing the UI was to me the most important part of all this.
  • Adding some animations, so that it would look a lot less boring.
  • Creating the entire biography game.
Let's just say the entire website. :)


What have i learnt?

I learnt how to code using more advanced techniques in HTML and CSS. Next to that how i learnt, how i could use the bootstrap library for a better responsive design. I also learnt how to design a good looking website. I learnt how to use Javascript to add animations to the pages. I also learnt to use Javascript and create a minigame with it, using the THREE.js library.
