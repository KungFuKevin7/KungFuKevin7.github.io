Green Water IoT


Assignment Info

Our Assignment was to create an IoT System that would be useful for any household. In the end we came up with the idea to create an automatic watering system for plants. The idea was simple create an app that interacts with an embedded system and water the plants with one button press. This was one of my personal favourite projects to work on.


My tasks

My main task was to create the app, the front-end of the application was my responsibility. Another task of mine was, to create an user-friendly interface, so that our system was very easy to use. I was also involved in the creation of the circuit and in writing the code for this circuit.


What have i learnt?

In this project i learnt how to create an application using Xamarin.Forms. I also delve into designing user-friendly Interfaces. On the back-end side, i learnt how to establish a connection between a microcontroller(Arduino) and a smartphone application Lastly, i learnt how to build circuits and write code for circuits in Arduino.
