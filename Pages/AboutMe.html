My Story


I was born in Heerenveen to Indonesian parents. I've lived all my life in the Netherlands and am able to speak Dutch and English fluently. In my early years, i was very intrigued in technology (mainly computers) and aviation. Every single day i sat behind the computer, browsing the internet for hours. I was also addicted to the gaming world and wanted to know more about the 'magic' behind video games and the internet. In 2018 i made the decision to follow the ICT trajectory. I wasn't the most experienced programmer compared to the other students, but i was motivated enough to carry on. After two years of following Computer Science, i had no doubt in my mind that i wanted to continue in the software engineering department. In 2020 i successfully finished secondary school and i didn't stop from there.

In the summer of 2020, i started to study HBO-ICT at NHL Stenden Leeuwarden. There i've learnt the fundamentals of programming and used that information to create applications, for school assignments and for fun to further improve my programming skill. I didn't start programming because school told me to, i started programming because i loved doing it. Currently i am in my third year and i'm not thinking about quitting anytime soon. In my free time i enjoy making music, coding, photography, playing video games, drawing and watching sports on television (mainly formula 1 and football).

Programming Experience


  • C#

    C# was the second programming language i ever learnt (my first one being python). I learnt C# during my first year of college and i cannot recommend it enough! It's my personal favourite language, thanks to it being very easy to understand and follow. It also has lots of availible frameworks, that are quite easy to use. I use C# a lot along with ASP.NET, Xamarin.Forms or WPF. And it's awesome!

  • HTML (+ CSS, Javascript)

    HTML was the first coding language i ever used. I really enjoy coding with html. It is probably the ('non-programming') language i know most of. I'm very fond of developing websites, because i just love to try different things with the website until all pixels look perfect. To add icing to the cake, html combined with CSS and javascript. I love working with CSS and javascript, because it brings the website to life. I adore seeing me very basic html website, turn into a living thing thanks to the power of CSS and Javascript.

  • Java

    Java was the first language i learned during my second year in college. I really enjoy programming with Java, mainly because it's very similar to C#. However my preference still goes to C#. My first ever project done in Java, was a raytracer without using any external libraries.

  • Python

    Python was my first ever programming language i learnt, but after that i honestly barely ever used it. It's not my favourite programming language, but it has things i like such as: the simplicity, and the short amount of code i have to write. However personally to me, the code is sometimes hard to follow. It primarily has to do with the fact that python uses indents instead of curly brackets, which i'm more used to. I mainly used it during my second year, to program a script that would convert controller output, into robot movement.
